## Multiple

`type="multiple"`로 설정하면 여러 항목을 동시에 열 수 있습니다.

```tsx
<Accordion type="multiple">
  <AccordionItem value="item-1">
    <AccordionTrigger>제목</AccordionTrigger>
    <AccordionContent>내용</AccordionContent>
  </AccordionItem>
</Accordion>
```

## Props

### Accordion

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `type` | `"single" \| "multiple"` | `"single"` | 단일/복수 열림 모드 |
| `defaultValue` | `string[]` | `[]` | 초기에 열려있을 항목 |

### AccordionItem

| Prop | Type | Description |
|------|------|-------------|
| `value` | `string` | 항목 고유 식별자 (필수) |

### AccordionTrigger / AccordionContent

별도 Props 없이 children만 전달합니다.
