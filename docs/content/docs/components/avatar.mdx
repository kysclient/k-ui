## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"sm" \| "md" \| "lg" \| "xl"` | `"md"` | 아바타 크기 |
| `src` | `string` | - | 이미지 URL |
| `alt` | `string` | - | 대체 텍스트 |
| `fallback` | `string` | - | 이미지 없을 때 표시할 텍스트 |
