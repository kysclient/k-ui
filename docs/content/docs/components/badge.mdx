## With Dot

`dot` prop을 사용하면 좌측에 상태 인디케이터가 표시됩니다.

<ComponentPreview>
  <div className="flex flex-wrap items-center gap-2">
    <Badge variant="success" dot>접수완료</Badge>
    <Badge variant="warning" dot>처리중</Badge>
    <Badge variant="error" dot>반려</Badge>
    <Badge variant="info" dot>공지</Badge>
  </div>
</ComponentPreview>

```tsx
<Badge variant="success" dot>접수완료</Badge>
<Badge variant="warning" dot>처리중</Badge>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "success" \| "warning" \| "error" \| "info" \| "neutral" \| "outline"` | `"default"` | 배지 스타일 |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | 배지 크기 |
| `dot` | `boolean` | `false` | 좌측 도트 인디케이터 |
