## Props

### Card

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "elevated" \| "outlined" \| "interactive"` | `"default"` | 카드 스타일 |
| `padding` | `"none" \| "sm" \| "md" \| "lg"` | `"md"` | 내부 여백 |

### Sub-components

| Component | Description |
|-----------|-------------|
| `CardHeader` | 상단 헤더 영역 |
| `CardTitle` | 제목 (`h3`) |
| `CardDescription` | 부제목/설명 |
| `CardContent` | 본문 콘텐츠 |
| `CardFooter` | 하단 액션 영역 (상단 보더 포함) |
