## Error State

`errorMessage`를 전달하면 자동으로 에러 스타일이 적용되고, `role="alert"`로 스크린 리더에 전달됩니다.

<ComponentPreview>
  <div className="w-full max-w-sm">
    <Input label="전화번호" placeholder="010-0000-0000" errorMessage="올바른 전화번호 형식이 아닙니다." required />
  </div>
</ComponentPreview>

```tsx
<Input
  label="전화번호"
  placeholder="010-0000-0000"
  errorMessage="올바른 전화번호 형식이 아닙니다."
  required
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | - | 라벨 텍스트 |
| `inputSize` | `"lg" \| "md" \| "sm"` | `"md"` | 입력 필드 크기 |
| `helperText` | `string` | - | 도움말 텍스트 |
| `errorMessage` | `string` | - | 에러 메시지 (설정 시 에러 스타일 자동 적용) |
| `required` | `boolean` | `false` | 필수 입력 표시 |
| `disabled` | `boolean` | `false` | 비활성 상태 |
| `readOnly` | `boolean` | `false` | 읽기 전용 |
