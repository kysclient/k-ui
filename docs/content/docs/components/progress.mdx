## Indeterminate

`indeterminate` prop을 사용하면 진행률을 알 수 없을 때 애니메이션이 표시됩니다.

<ComponentPreview>
  <div className="w-full max-w-md space-y-4">
    <Progress indeterminate />
    <Progress indeterminate variant="success" />
  </div>
</ComponentPreview>

```tsx
<Progress indeterminate />
<Progress indeterminate variant="success" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | `0` | 현재 진행률 (0-100) |
| `max` | `number` | `100` | 최대값 |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | 바의 높이 |
| `variant` | `"default" \| "success" \| "warning" \| "error"` | `"default"` | 색상 변형 |
| `showValue` | `boolean` | `false` | 퍼센트 텍스트 표시 |
| `indeterminate` | `boolean` | `false` | 불확정 애니메이션 상태 |
| `className` | `string` | - | 추가 CSS 클래스 |

## Accessibility

- `role="progressbar"` 속성이 자동으로 적용됩니다.
- `aria-valuenow`, `aria-valuemin`, `aria-valuemax` 속성이 설정됩니다.
- `indeterminate` 상태에서는 `aria-valuenow`가 제거되어 스크린 리더에 불확정 상태임을 알립니다.
- 색상만으로 상태를 전달하지 않도록 `showValue`와 함께 사용하는 것을 권장합니다.
