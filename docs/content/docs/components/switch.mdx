## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | - | 라벨 텍스트 |
| `checked` | `boolean` | - | 제어 모드 값 |
| `defaultChecked` | `boolean` | `false` | 초기 값 |
| `disabled` | `boolean` | `false` | 비활성 상태 |
| `onChange` | `function` | - | 변경 이벤트 핸들러 |
