```tsx
<Tooltip content="도움말 텍스트" side="top">
  <Button>호버하세요</Button>
</Tooltip>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `content` | `ReactNode` | - | 툴팁 내용 (필수) |
| `side` | `"top" \| "bottom" \| "left" \| "right"` | `"top"` | 표시 위치 |
| `delayMs` | `number` | `200` | 표시 지연 시간(ms) |
